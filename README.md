# CSVault

A Discord bot for viewing all Counter-Strike information regarding skins, cases, collections, and more.

## Features
- Live item info and prices from database and API
- Pokedex style inventory system to track user inventory
- Commands ran directly inside Discord to avoid redundant Web searches

## Slash Commands
- `/agent {name}` to view a Agent.
- `/case {name}` to view a Case.
- `/collection {name}` to view a Collection.
- `/package {name}` to view a Package.

## Tech Stack
- Interactions.py
- PostgreSQL
- Discord API
- Skins.GG API
